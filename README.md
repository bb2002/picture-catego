PICTURE CATEGO 프로그램에 오신것을 환영합니다.

[HOW TO USE]
1. 패키지를 다운로드 받습니다.
2. npm install 또는 yarn 을 통해 의존성 파일을 설치합니다.
3. app.js 파일이 있는 위치에 ./images 폴더와 ./output 폴더를 만듭니다.
4. ./images 폴더에 이미지를 추가합니다. (images 아래 폴더를 만들어서는 안됩니다.)
5. 명령 프롬프트를 통해 "nodejs app.js" 명령을 실행합니다.

[기능]
- 사진의 EXIF 데이터를 읽어 YYYY. MM 형태의 이름으로 구성된 폴더에 정리합니다.
- EXIF 데이터가 없는 경우, 사용자에게 직접 입력을 받습니다.
- 사용자도 이 파일의 날짜를 모르는 경우, 해당 파일이 생성된 날짜를 기준으로 합니다.
- ./output 폴더의 pictures 폴더에는 사진이, videos 폴더에는 동영상이 저장됩니다.

[주의]
- ./images 폴더는 반드시 생성되어있어야 합니다.
- ./output 폴더는 반드시 비워져있어야 합니다.
- 필요에 따라 자유롭게 수정 할 수 있습니다.

[용도]
- NAS 파일 정리 (개발자는 이것을 위해 만듦)
